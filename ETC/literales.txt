1.-Introducción breve y contextualización - 25% de la nota del ejercicio

En este ejercicio se desarrolla una pequeña aplicación web utilizando literales en JavaScript. Los literales son valores fijos que se escriben directamente en el código
como cadenas de texto, números, booleanos y arreglos. Nuestro objetivo es comprender cómo se declaran y utilizan estos valores para representar información dentro de una página web.

2.-Desarrollo detallado y preciso - 25% de la nota del ejercicio

En este codigo ejemplo de codigo basico de JavaScript usando literales, podemos observar que el literal que se usa es de tipo string, que se define como string al principio del codigo
usando "", asi:

```
var artista_favorito= ""

```
Este seria nuestro literal en el codigo que no sdice cual es el artista favorito del usuario (Jose Vicente Carratala). Luego procedemos a asignar el valor al literal que seria "Coldplay" poniendolo
dentro de los parentesis y comillas (porque es string)asi:

```
artista_favorito = ("Coldplay")

```
Este literal almacenan el dato de cual es el artista favorito en el codigo y a continuacion, se muestra en el documento usando document.write seguido de el literal entre parentesis, tal que asi:
```
document.write(aritsta-favorito)

```
